# System analysis
Учебные проекты по системному анализу, включающие в себя: 
- описание контекста систем "Автоматизированный гардероб", "сушилка для обуви"
- Моделирование IDEF, ER, BPMN диаграмм для "киберклуба"
- Выпускную квалификационную работу по специальности "Системный анализ", которая содержит описание контекста и архитектуры системы на уровне операционного анализа, системного анализа и на логическом уровне
